# プログラムの紹介
これは顔を動かして、クリアを目指すゲームです。
プログラム内で使用される画像や音楽は、任天堂マリオブラザーズで使用されているものを使用しました。＊1
プログラムは鼻を認識します。
鼻には選択したキャラクターが表示されます。
左から来るドカンに当たらずに指定数のドカンをよければゲームはクリアです。

# 操作方法
顔を認識できなければゲームにならないので、なるべく部屋の明かりを明るくして、PCの画面も明るくしてください。
初めにキャラクター選びスタートします。
ゲームオーバー或はクリアしたあ後にキーを押すとリスタートできます。

# 画面遷移
画面の種類は下記の四つがあります。
・スタート画面
・ゲームオーバー画面
・クリア画面
・内カメからの映像
